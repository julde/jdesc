the pd examples are really preliminary now. the main file to start with is GUI_Example.pd which makes use of most of the abstractions in the examples folder. they have been developed using pd 0.38.4-rc8 extended and may require externals not included in the basic pd installation (e.g. send13 and the OSC classes).

please see the HELP subpatch inside the GUI patch to learn a bit about the concept. feel free to contact me if you are interested in developing GUI "classes" for pd.
